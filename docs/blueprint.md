# **App Name**: FlowSnap

## Core Features:

- Secure User Authentication: Allow users to securely sign up and log in using email/password or Google, protecting their financial data.
- Multi-Account Book Management: Enable users to create, view, and manage multiple financial 'books' or accounts, each with a distinct name and creation date.
- Book Overview Dashboard: Provide a clear, summarized dashboard for each financial book, displaying the net balance, total cash in, and total cash out.
- Detailed Transaction History: Display a scrollable, mobile-optimized list of all transactions within a book, showing method, value, running balance, category, and timestamp.
- Quick Transaction Entry: Allow users to quickly record 'Cash In' or 'Cash Out' transactions via an intuitive modal form, automatically updating account balances upon submission.
- Transaction Filtering System: Enable users to efficiently filter their transactions by method (Cash/Online), category, and a custom date range for better financial insights.
- Smart Category Suggestion: An AI tool that suggests relevant categories for new transactions based on keywords from the transaction details or common spending patterns, enhancing data entry speed and accuracy.

## Style Guidelines:

- The primary color is a deep, muted violet-blue (#62598F), chosen to convey trustworthiness and sophistication for financial management. It evokes a sense of calm reliability.
- The background color is a very light, cool gray-violet (#F4F3F7). This highly desaturated hue maintains coherence with the primary color while providing a clean, airy canvas for content in a light theme.
- The accent color is a vibrant sky blue (#61A3ED). Positioned analogously on the color wheel relative to the primary, its brighter, more saturated nature creates impactful highlights for interactive elements and call-to-actions, providing visual contrast and energy.
- The application will utilize 'Inter' for both headlines and body text. As a grotesque-style sans-serif, 'Inter' offers a modern, neutral, and highly legible appearance optimized for user interfaces, ensuring clarity for numerical data and transaction details.
- Implement a set of minimalist line-art icons that are consistent in style. Use familiar symbols for navigation, transaction types (e.g., up/down arrows for cash in/out), and account management (e.g., a ledger book icon).
- A mobile-first layout prioritizing vertical content stacks, clear information hierarchy, and ample whitespace. Dashboard elements and transaction lists will be presented within card-like structures to improve readability and visual separation on small screens.
- Incorporate subtle and functional animations to enhance user feedback, such as smooth transitions for modals, gentle loading indicators for data fetches, and discrete confirmations upon successful transaction entries.
